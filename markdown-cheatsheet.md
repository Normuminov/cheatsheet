
# Markdown cheatsheet

There are 6 sizes of headers:

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

## Text formatting:

Normal text

**Bold text**

_Italic text_

## Links
### _Inline link:_

![Link](red-apple.jpg)

### _Reference link_:

[Click here!][link]

[link]: www.example.com

## Images:

### Inline image

![Screenshot](yellow-apple.jpg)

### Reference image

![Green apple][Green apple]

[Green apple]: green-apple.jpg
